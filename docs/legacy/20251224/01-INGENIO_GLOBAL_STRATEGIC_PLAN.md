## 核心愿景 (The Vision)

**Ingenio Global** 是一个集 **"AI Salesforce (云端算力)" + "Autonomous Factory (自治工厂)" + "Global Guild (全球公会)"** 于一体的超级生态系统。

我们的目标是打造**应用层的 Agent 孵化器**。
让任何个人或团队，通过**Vibe Coding (灵感交互)**，利用**组件化市场**的能力，一站式孵化出具备**统一记忆**、**抗封号加密**的全球化 App 矩阵。

---

## 第一支柱：技术架构 (The Tech Stack)

### 1. L1: The Cloud Brain (AI Salesforce / 云端大脑)
*   **定义:** 这是一个集中的 AI 服务提供商 (Centralized AI Service Provider)。未来的每个 App 只是一个端点，核心能力通过 API 租用平台服务。
*   **Unified Memory (统一记忆):**
    *   **User Memory Agent:** 每个 App 内置记忆 Agent。用户的行为偏好在云端统一更新、跨应用同步（多模态向量存储）。
*   **Omni-Ready Kernel (全模态内核):**
    *   支持 **Vision-to-Code** (看图写代码)。
    *   预留 Omni 接口，随时集成未来的多模态大模型。

### 2. L2: The Autonomous Factory (自治工厂)
*   **Vibe Coding (灵感编程):**
    *   对标 **Devin**。用户只负责“定义产品形态和交互体验” (Define the Vibe)，Agent 负责自主执行、代码编写、自我修复。
*   **Polymorphic Security (多态加密):**
    *   **核心护城河:** 为了避免同类产品被平台 (App Store) 下架。
    *   **Code Encryption:** 对核心业务逻辑进行混淆加密。
    *   **Mutation:** 生成的代码在 AST 层面进行变异，确保每个 App 的二进制指纹独一无二。

### 3. L3: The Marketplace (组件化市场)
*   **Component Economy:** 技术能力组件化。开发者可以开发组件 (如: 3D 地球、PDF 引擎) 上传到平台。
*   **Tenant Isolation (多租户):**
    *   企业/公司可通过 **Private Tenant (私有租户)** 上传私有接口 (Private APIs) 和私有数据，构建企业级内部应用。

## 第二支柱：生态闭环 (The Ecosystem)

### 1. The Guild (全球公会)
*   **定位:** **海外版“生财有术”创作营**。
*   **功能:**
    *   **Community:** 创作者讨论、分享出海经验、组队打怪。
    *   **Knowledge Base:** 沉淀“如何用 Ingenio 做 SEO”、“如何做流量”的实战手册。
    *   **Resource Mart:** 交易 Prompt、UI 皮肤、商机线索。
*   **协同化 (Collaboration):** 支持团队作战 (Squad Mode)，多人协同管理一个 App 矩阵。

### 2. The Matrix (集群化)
*   **App Clustering:** 用户生成的不是单一 App，而是**产品矩阵** (e.g., 植物识别 -> 昆虫识别 -> 宠物识别)。
*   **Traffic Interconnection:** 矩阵内的 App 可以相互导流，形成流量闭环。

## 第三部分：商业模式 (Business Model)

我们采用分阶段演进的 **"D-B-C" 混合变现模型**：

### 阶段一：D2C 自营孵化 (Incubation Mode - Day 1 to Month 6)
*   **策略 (Dogfooding):** 团队自身作为“超级开发者”，利用平台孵化 C 端 App 矩阵。
*   **收入:** App Store 内购、Stripe 订阅流、广告费。
*   **目的:** 赚取早期现金流养活团队，同时在实战中打磨平台能力，积累成功案例 (Case Studies)。

### 阶段二：B2B SaaS 扩张 (Enterprise Mode - Month 6 to Year 2)
*   **策略 (Agency in a Box):** 将成熟的平台卖给 B 端（外包公司、创业工作室）。
*   **产品:**
    *   **Pro License ($99/mo):** 适合独立开发者，提供标准模版。
    *   **Studio License ($499/mo):** 适合工作室，提供 AST 防封号编译器、团队协作。
    *   **Private Tenant:** 为企业提供数据隔离的私有租户环境。
*   **收入:** 稳定的 SaaS 订阅费 (ARR)。

### 阶段三：D->B->C 生态终局 (Ecosystem Mode - Year 2+)
*   **策略 (Marketplace):** 引入 D 端（技术开发者），形成 "Developers build Components -> Businesses build Apps -> Consumers use Apps" 的闭环。
*   **收入:**
    *   **API Toll:** 平台作为连接器，抽取 API 调用差价和“过路费”。
    *   **Marketplace Tax:** 组件与资源交易抽成 (20-30%)。
